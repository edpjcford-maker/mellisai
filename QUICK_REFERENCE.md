# Mellisai Cloudinary - Quick Reference Card

## 📋 Setup Checklist

```
□ Create account at cloudinary.com
□ Find Cloud Name in Dashboard
□ Create Upload Preset (Settings → Upload)
□ Set Upload Preset to "Unsigned"
□ Copy preset name
□ Open Mellisai in browser
□ Click "Cloudinary Settings" button
□ Paste Cloud Name
□ Paste Upload Preset name
□ Click "Save Settings"
```

---

## 🔑 Where to Find Your Info

| Info | Location |
|------|----------|
| Cloud Name | Cloudinary Dashboard (top center) |
| Upload Preset | Settings → Upload → Upload presets |
| API Key | Settings → Access Keys |

---

## 📝 Common Errors & Fixes

| Error | Fix |
|-------|-----|
| "Configure Cloudinary settings" | Enter Cloud Name in settings |
| Upload fails silently | Check Cloud Name spelling |
| "Preset not found" | Create unsigned preset, check name |
| No internet upload | Check internet connection |
| File too large | Keep files under 100MB |

---

## 🎯 Upload Best Practices

✅ **Do:**
- Use MP3 files (best compatibility)
- Keep files under 100MB
- Use descriptive file names
- Create organized presets

❌ **Don't:**
- Share API Secret key
- Use huge files (10GB+)
- Upload copyrighted content
- Forget to save settings

---

## 💾 File Formats Supported

✅ Audio: MP3, WAV, OGG, M4A, FLAC, AAC, OPUS
❌ Video: Not supported for this app

---

## 🔐 Security Reminder

- Settings stored **locally** only
- No passwords uploaded to servers
- Cloudinary is encrypted/secure
- Safe to use unsigned uploads
