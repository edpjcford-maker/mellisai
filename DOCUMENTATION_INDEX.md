## Common Questions

**Q: Which file should I read first?**
A: `QUICKSTART.md` - gets you running in 5 minutes

**Q: I'm stuck on setup**
A: Read `SETUP_WORKFLOW.md` step-by-step or check `CLOUDINARY_SETUP.md` troubleshooting

**Q: I forgot my Cloud Name**
A: Go to https://cloudinary.com/console - it's shown at top

**Q: How do I change Cloudinary settings?**
A: Click "Cloudinary Settings" button anytime in Mellisai

**Q: Where are my files stored?**
A: Cloudinary cloud servers (secure, permanent)

**Q: Can I download my uploaded songs?**
A: Yes, from Cloudinary Media Library
