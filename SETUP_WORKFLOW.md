# 📋 Mellisai - Step-by-Step Cloudinary Setup Guide

## Complete Workflow for First-Time Users

---

## Step 1: Create Cloudinary Account (5 minutes)

### 1.1 Visit Cloudinary
```
→ Open browser
→ Go to: https://cloudinary.com
→ Click "Sign Up Free"
```

### 1.2 Choose Plan
```
→ Click "Free Plan" (perfect for music)
→ Fill in email, password
→ Check the box "I agree to the terms"
→ Click "Sign Up"
```

### 1.3 Verify Email
```
→ Check your email inbox
→ Find verification email from Cloudinary
→ Click the verification link
→ Done! Account is ready
```

---

## Step 2: Find Your Cloud Name (2 minutes)

### 2.1 Log In to Dashboard
```
→ You'll be taken to Cloudinary Dashboard
→ (If not, go to: https://cloudinary.com/console)
```

### 2.2 Locate Cloud Name
```
→ Look at the top-center of the Dashboard
→ You'll see something like:
   "Cloud Name: your-cloud-123xyz"
→ Copy this text (just the part after the colon)
→ Example: your-cloud-123xyz
```

### 2.3 Save It
```
→ Keep this handy (you'll need it in 5 minutes)
→ Or write it down
```

---

## Step 3: Create Upload Preset (3 minutes)

### 3.1 Open Settings
```
→ Click "Settings" (gear icon, bottom left)
→ Settings page opens
```

### 3.2 Go to Upload Tab
```
→ Click "Upload" tab
→ Scroll down to find "Upload presets"
```

### 3.3 Create New Preset
```
→ Click "Add upload preset"
→ Fill in:
   - Name: mellisai (or your choice)
   - Unsigned: Toggle to ON (blue)
   - Resource type: Select "Video" or "Auto"
→ Click "Save"
```

### 3.4 Copy Preset Name
```
→ You'll see your new preset in the list
→ Copy the preset name
→ Example: mellisai
```

---

## Step 4: Open Mellisai App (1 minute)

### 4.1 Open HTML File
```
→ Open index.html in your browser
→ Mellisai music player loads
→ You see 15 pre-loaded songs
```

---

## Step 5: Configure Mellisai Settings (2 minutes)

### 5.1 Click Settings Button
```
→ In the left sidebar
→ Find "Cloudinary Settings" button
→ Click it
→ Modal popup appears
```

### 5.2 Fill in Cloud Name
```
→ Find the "Cloud Name:" field
→ Paste your Cloud Name
→ Example: your-cloud-123xyz
```

### 5.3 Fill in Upload Preset
```
→ Find the "Upload Preset:" field
→ Paste your preset name
→ Example: mellisai
→ Leave API Key empty (optional)
```

### 5.4 Save Settings
```
→ Click "Save Settings" button
→ Success! You see green notification
→ Modal closes automatically
```

---

## Step 6: Upload Your First Song (2 minutes)

### 6.1 Click Upload Button
```
→ In the left sidebar
→ Click "Upload Song" button
→ File picker opens
```

### 6.2 Select Audio File
```
→ Browse your computer
→ Select an MP3 or audio file
→ Click "Open"
```

### 6.3 Watch Upload
```
→ Progress modal appears
→ You see percentage (0% → 100%)
→ Status messages show progress
```

### 6.4 Song Added
```
→ Upload completes
→ Success notification appears
→ "Play this song now?" prompt
→ Click YES to play, NO to skip
```

---

## Step 7: Enjoy! (∞)

```
→ Your song is now in the playlist
→ Click any song to play it
→ All controls work (play, skip, volume, etc.)
→ Song plays from Cloudinary
→ Upload more songs anytime!
```

## 🆘 Troubleshooting During Setup

### Problem: Can't find Cloud Name
**Solution:**
- Go to https://cloudinary.com/console
- Log in with your account
- Look at top-center of page
- It's shown right there

### Problem: Upload preset not found error
**Solution:**
- Make sure preset is set to **"Unsigned"** (very important!)
- Double-check spelling of preset name
- Try creating a new preset with simpler name

### Problem: "Settings not saved" message
**Solution:**
- Make sure Cloud Name field is not empty
- Click "Save Settings" button again
- Check browser console (F12) for errors

### Problem: Upload fails silently
**Solution:**
- Verify Cloud Name spelling (case-sensitive)
- Check file is under 100MB
- Try with different audio file
- Check internet connection

## ✨ Pro Tips

**Tip 1: Organize Files**
- Before uploading, put songs in a folder on your computer
- Makes it easier to select multiple files

**Tip 2: Use Descriptive Names**
- Rename songs before uploading
- Makes them easier to find later
- Example: "Roja Ondru - Ilaiyaraaja.mp3"

**Tip 3: Batch Upload**
- Drag multiple files into Mellisai
- Upload several at once
- Faster than one-by-one

**Tip 4: Test First**
- Try uploading a small file first
- Verify it works
- Then upload your whole library

**Tip 5: Bookmark Cloudinary**
- Bookmark https://cloudinary.com/console
- Easy access to your media library
- Manage uploads from there too
