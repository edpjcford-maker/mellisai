# Mellisai - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Open the App
```
1. Open index.html in your web browser
2. You'll see Mellisai with 15 pre-loaded songs
```

### Step 2: Test Basic Features
```
✓ Click any song to play it
✓ Use play/pause buttons
✓ Adjust volume slider
✓ Click progress bar to seek
✓ Try keyboard spacebar for play/pause
```

### Step 3: Set Up Cloudinary (Optional but Recommended)
```
1. Click "Cloudinary Settings" button in the sidebar
2. Create free account at cloudinary.com
3. Enter your Cloud Name
4. Create an unsigned upload preset
5. Enter the preset name
6. Click "Save Settings"
```

### Step 4: Upload Your First Song
```
1. Click "Upload Song" button
2. Select an audio file (MP3, WAV, OGG, etc.)
3. Wait for upload to complete
4. Song automatically added to playlist!
```

### Step 5: Enjoy!
```
✓ Songs now in your Mellisai library
✓ All features ready to use
✓ Settings saved in browser
```

## 🎵 Now Playing Features

- **Play/Pause**: Click center play button
- **Next/Previous**: Use arrow buttons  
- **Shuffle**: Click shuffle icon to randomize
- **Repeat**: Click repeat icon to cycle modes
  - 1st click: Repeat all
  - 2nd click: Repeat one
  - 3rd click: No repeat
- **Volume**: Drag volume slider
- **Seek**: Click progress bar to jump to time

---

## 📱 Mobile Tips

- **Menu**: Tap hamburger icon (☰) to open sidebar
- **Close Menu**: Tap X button or click song
- **Touch Controls**: All buttons are touch-friendly
- **Landscape**: Works great in landscape mode
- **Portrait**: Optimized for portrait viewing

---

## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| No sound | Check volume isn't muted, try another song |
| Upload fails | Set up Cloudinary settings first |
| Sidebar won't close | Tap the X button or click a song |
| Settings disappeared | Browser storage was cleared, re-enter them |
| Song won't play | Check browser console (F12) for errors |
